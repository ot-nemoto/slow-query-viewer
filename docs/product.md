# プロダクト定義

## 目的

MySQL のスロークエリログファイルをブラウザ上で解析・可視化し、パフォーマンスボトルネックの特定を支援するツール。

コマンドラインツールや DB 専門知識がなくても、ログファイルをドラッグ＆ドロップするだけでクエリの実行傾向を把握できる。

## 対象ユーザー

- MySQL を運用するデータベース管理者
- アプリケーション開発者（パフォーマンス調査時）
- DevOps / SRE エンジニア

## 成功指標

- スロークエリログをアップロードするだけで、クエリの統計・時系列・パラメータ分析が即座に確認できる
- 複数ファイルを統合して比較分析できる
- サーバー不要（クライアントサイド完結）で、GitHub Pages 上で誰でも利用できる

## 主要機能

- **ファイル管理**: 複数ファイルの同時アップロード、ドラッグ＆ドロップ、個別/一括削除
- **ログ解析**: MySQL スロークエリログ形式（AWS RDS 形式含む）の解析、不正エントリのスキップ
- **クエリ正規化・グルーピング**: リテラル置換による正規化、パラメータ値別の詳細分析
- **統計サマリー**: 総クエリ数、実行時間の各指標、最遅クエリ詳細、解析期間
- **結果テーブル**: 正規化クエリ別の集計、各列ソート、上位 20 件表示
- **時系列チャート**: ファイル別の表示切り替え、ツールチップでの詳細確認
- **通知**: トースト通知（成功/警告/エラー）、5 秒自動消去
